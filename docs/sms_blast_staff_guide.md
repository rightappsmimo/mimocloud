# SMS Blast Guide (Non-Technical)

## What is SMS Blast?

SMS Blast lets you send text messages to parents and guardians at Mimo Play Cafe. You can:
- Send messages to all parents at once
- Send to specific parents
- Schedule messages for later
- Let the system send automatic reminders

## Getting Started

### Accessing SMS Blast

1. Log into the Admin Panel
2. Click on "SMS Blasts" in the menu
3. You'll see the main dashboard with:
   - **Total** - All messages ever sent
   - **Sent** - Messages completed successfully
   - **Scheduled** - Messages waiting to be sent
   - **Failed** - Messages that had problems

## Creating a Message

### Step 1: Click "Create New Blast"

### Step 2: Fill in the Details

**Title**: Give your message a name (for your reference)

**Type**: 
- **Campaign** - For manual messages you create
- **Automation** - These are automatic messages (don't change this)

**Message**: Type your text here (max 255 characters)

### Step 3: Use Templates

Instead of writing from scratch, you can use ready-made templates:
- **Birthday Greetings** - For children's birthdays
- **Time is Almost Up** - 10-minute warning before session ends
- **Overtime** - When playtime is exceeded
- **Check Out** - Session end reminder

### Step 4: Personalize Your Message

Use these codes to add names automatically:
- `{child_name}` - Child's name appears here
- `{parent_name}` - Parent's name appears here

Example: "Hi {parent_name}, {child_name}'s session ends soon!"

### Step 5: Choose When to Send

**Send Mode**:
- **Now** - Send immediately
- **Scheduled** - Pick a date and time to send later
- **Every Time** - Don't use this (for automatic messages only)

### Step 6: Choose Who Gets It

**Recipients**:
- **All Contacts** - Send to everyone
- **Search & Select** - Pick specific parents

## Automatic Messages

The system sends these automatically - no setup needed:

| Message | When It Sends |
|---------|---------------|
| Birthday Greetings | Child's birthday at 9:00 AM |
| 10-Minute Warning | 10 minutes before play session ends |
| Checkout Reminder | When session time is up |
| Overtime Alert | When child stays past their time |

## Managing Your Messages

### View Message History
- Click the eye icon to see who received your message
- See how many succeeded vs failed

### Resend Failed Messages
If some messages fail:
1. Find the message in the list (status: Failed)
2. Click the circular arrow button
3. The system will try again

### Edit or Delete
- **Edit**: Available for drafts and automatic messages
- **Delete**: Only for drafts (click trash can)

## Common Questions

### Why did my message fail?
- Parent's phone number might be missing or wrong
- Check the error message in the recipient list

### Can I send picture messages?
Not currently - only text messages (SMS).

### What's the character limit?
255 characters maximum (about 4-5 short sentences).

### Can I cancel a scheduled message?
Yes, delete it while it's still in draft status.

### How do I know my message was received?
- Green "Sent" count shows successful deliveries
- Check the detailed list for individual status

## Best Practices

1. **Keep it short** - Under 160 characters for single SMS
2. **Use names** - Messages with names get better responses
3. **Check before sending** - Make sure spelling is correct
4. **Monitor failures** - Resend if needed
5. **Time zone** - Messages follow Manila time (PHT)
